How to run:
1. Install and run Memcached per instructions at /Memcached/Instructions.txt.
2. Start and run web2py and rocket
	2a Open a terminal and cd to web2py folder
	2b Run command: web2py.exe -i 127.0.0.1 -p 8000 -a 'x' -k mysite.key -c mysite.crt
	2c Navigate to https://127.0.0.1:8000
	2d Log in as one of the users
